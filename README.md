This library is designed with:
- **Performance** in mind ‚Äì low allocations, cache-friendly.
- **Concurrency safety** (optional) ‚Äì safe variants for multi-goroutine workloads.
- **Ease of use** ‚Äì clean API, well-documented, and consistent.

## ‚ú® Features

- Common collections:
  - `Stack`
  - `Queue`
  - `Deque`
  - `LinkedList` (singly & doubly linked)
  - `HashMap` (chained & open addressing)
  - `Set` (hash-based & tree-based)
- Tree structures:
  - `Binary Search Tree`
  - `AVL Tree`
  - `Red-Black Tree`
  - `Trie`
- Graph representations:
  - Adjacency list
  - Adjacency matrix
- Priority structures:
  - `Heap` (min & max)
  - `PriorityQueue`
- Thread-safe variants with `sync.Mutex` or `sync.RWMutex`.
- Custom iterators for all collections.
